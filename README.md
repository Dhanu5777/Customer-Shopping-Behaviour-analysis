# Customer Shopping Behaviour Analysis

## Project Overview
This project focuses on analyzing customer shopping behaviour to understand purchasing patterns, customer demographics, product performance, and payment preferences.  
The goal of the project is to provide meaningful insights that help businesses make data-driven decisions.

## Tools & Technologies Used
- Python (Pandas, NumPy)
- SQL
- Power BI

## Project Workflow
1. Performed data cleaning and preprocessing using Python
2. Handled missing values and created new features such as age groups and purchase frequency
3. Stored and analyzed data using SQL queries
4. Built an interactive Power BI dashboard to visualize insights

## Key Insights
- Certain age groups purchase more frequently than others
- Some product categories generate higher sales
- Digital payment methods are preferred over cash
- Discounts and promotions increase purchase frequency
- Repeat customers indicate customer loyalty

## Files in This Repository
- customer_shopping_behaviour.ipynb : Python notebook for data cleaning and analysis
- customer_shopping_behaviour.sql : SQL queries used for analysis
- Customer behavior dashboard.pbix : Power BI dashboard file

## Conclusion
This end-to-end data analytics project demonstrates the use of Python, SQL, and Power BI to transform raw customer data into actionable business insights.
